## 이미지 권장 사양

- **형식**: JPG (사진), PNG (로고)
- **해상도**: 최소 1920x1080px (Full HD)
- **크기**: 각 파일 500KB 이하 권장
- **비율**: 
  - 배경 이미지: 16:9 또는 21:9
  - 제품 스크린샷: 16:9
  - 활용 케이스: 4:3 또는 16:9

## 참고

이미지가 없어도 웹사이트는 정상 작동합니다. 이미지가 없을 경우 자동으로 대체 이미지나 플레이스홀더가 표시됩니다.
